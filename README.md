# 💰 Expense Tracker App (Data Science + ML Project)

## 📌 Overview

The **Expense Tracker App** is a data-driven web application built using Python and Streamlit. It allows users to record, analyze, and visualize their expenses while leveraging Machine Learning to predict future spending patterns.

This project demonstrates real-world applications of **Data Analysis, Visualization, and Predictive Modeling** in personal finance — similar to features used in modern FinTech platforms.

---

## 🎯 Problem Statement

Managing personal expenses manually is inefficient and lacks insights. Users often:

* Don’t know where money is being spent
* Cannot track monthly trends
* Fail to control overspending

---

## 💡 Solution

This project provides:

* A centralized expense tracking system
* Automated data analysis
* Interactive dashboards
* ML-based future expense prediction

---

## 🚀 Features

### ✅ Core Features

* Add and store expenses
* Category-wise expense tracking
* Monthly trend analysis
* Clean and interactive dashboard

### 📊 Data Visualization

* Pie chart → Category distribution
* Line chart → Monthly spending trend
* KPI cards → Total, average, transactions

### 🤖 Machine Learning

* Linear Regression model
* Predicts next 7 days of expenses
* Helps in financial planning

### ⚠️ Alerts

* Detects high spending behavior

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/Expense-Tracker-App.git
cd Expense-Tracker-App
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```bash
python -m pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
python -m streamlit run app.py
```

## ⚠️ Troubleshooting

```bash

```

**3. Empty dataset error**

* Delete `data/expenses.csv`
* Restart app
